# Задание: Нейронная сеть прямого распространения — классификация

Примените нейронную сеть прямого распространения для задачи классификации.
Выберите любой реалистичный датасет для классификации и обучите модель, а затем получите предсказания.
Сравните качество с логистической регрессией и сделайте вывод по метрикам.
